Sealink

Decentralized LinkTree on Sui + Walrus

  • Own your profile and links
  • End‑to‑end encrypted, anonymous messaging
  • NFT gallery and marketplace

The problem

  • Centralized link tools are fragile and siloed
  • Creators risk losing reach and data
  • Audiences expect privacy when contacting creators

The solution

  • On‑chain profile anchored on Sui
  • Walrus blobs for fast, portable content
  • Encrypted, anonymous messages with on‑chain references

Architecture

Smart contracts

Move modules for profiles, links, message refs

Storage

Walrus content‑addressed blobs for media and messages

Frontend

React + Vite, Mysten dapp‑kit, Walrus APIs

On‑chain Profiles

  • Username, bio, avatar, themed links
  • Walrus blob reference for cross‑device reads
  • Portable and update‑friendly

Sealed Messaging

  • Anonymous or named messages
  • Client‑side encryption, Walrus storage
  • Only references on‑chain

NFTs & Marketplace

  • Media and metadata stored via Walrus
  • Free drops and paid items
  • Wallet‑native purchase flows

Developer view

  • Move contracts, Sui dapp‑kit
  • Walrus upload/fetch API, batch fetch
  • ENV flags for package/registry IDs

User journey

  • Connect wallet → create profile
  • Visitors send sealed messages
  • Mint and showcase NFTs

Security & Privacy

  • On‑chain: identifiers and references
  • Walrus: encrypted payloads and media
  • Production: audited crypto and key mgmt

Roadmap

  • Harden cryptography + key management
  • Full on‑chain NFT mint flows
  • Moderation tooling and reporting

Call to action

  • Try the demo
  • Share feedback
  • Contribute on-chain and frontend
